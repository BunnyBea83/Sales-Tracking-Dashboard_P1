# Contributors to Sales Tracking Dashboard

We are happy to have our team members contribute to the analysis of our developing company. There are many ways in which this project was created. Contributions include, data refinement, data visualization, and forcasting

## Contributing Code:

General workflow for contributing to this project

* Fork the repository
* Create a local branch to add contributions
* Commit your changes and push your created branch to your fork
* Open a new pull request into our master branch
